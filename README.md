# `CuriosityBot-MD`

### ACTIVA EL TERMUX
```sh
$ apt-get update -y && apt-get upgrade -y
$ pkg install -y git nodejs ffmpeg imagemagick && pkg install yarn
$ git clone https://github.com/Azami19/CuriosityBot-MD1 && cd CuriosityBot-MD1 && npm install && yarn 
$ npm start
```
